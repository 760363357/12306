# 12306
## 本项目为12306购票脚本

注：暂时无法识别验证码，需要手动输入验证码的位置信息


文档目录说明：

- 主目录

    - buy_ticket      //该目录用于购票的函数
    
            __init__.py   // 初始化一些需要的变量
            buy.py       // 买票函数的定义
            login.py     // 登录类定义
            vaild_image.py  //验证类定义
    - query
    
            __init__.py   
            query_tick.py  //查询车票信息并根据要求提取出一条车票信息的定义
            ticket.py   //获取一条车票信息的类定义
            station.csv  //所有城市的代码文件
    - 12306.png             //12306验证码图片
    - buy_tick.py           //主文件，直接执行即可
    - config                //项目配置信息


使用：

1.先根据config里面的配置信息提示填写配置信息

2.直接执行buy_tick.py

3.根据提示输入验证码坐标信息，直接输入数字1-8即可（从左往右数）,输入一个数字回车一次，输入'q'即完成输入，输入'r'刷新图片

4.等待程序自动执行，正常执行即可完成购票。
